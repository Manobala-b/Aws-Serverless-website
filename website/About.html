About CloudForge

CloudForge is a serverless web application project designed to showcase how to build a secure, scalable, and cost-effective website using AWS services. It demonstrates the use of Amazon S3 for static hosting, Lambda for backend logic, API Gateway for secure APIs, DynamoDB for data storage, and CloudFront for global content delivery.

This project provides an excellent foundation for students and developers to understand cloud-native architectures and serverless technologies. CloudForge eliminates the need for traditional servers, making it lightweight, fast, and easy to maintain.
